## Current Integration Architecture

### Existing Systems
- **VinVideo Connected**: 4-agent pipeline (Producer, Director, DoP, Prompt Engineer) → FLUX images
- **Editing Project**: Professional video editing system with RunPod worker + S3 integration

### Data Flow Design
```
VinVideo Pipeline → Generate Images → Upload to S3 → Editing Agent → RunPod → Final Video
```

### S3 Folder Structure (Established)
```
s3://vinvideo/
├── input/project-{id}/
│   ├── beat_1.png, beat_2.png, ... (images from VinVideo)
│   ├── audio.wav (TTS voiceover)
│   ├── producer.json, director.json, etc. (agent outputs)
│   └── editing_plan.json (generated by editing agent)
└── output/project-{id}/
    └── final_video.mp4 (professional edited video)
```
